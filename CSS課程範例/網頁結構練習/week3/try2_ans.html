Photoshop Web 網頁版型設計專修班

Photoshop 是目前網頁設計界最重要的軟體，想學習職業等級的網頁版型技巧以及網頁構圖嗎? 您不能錯過本教學課程喔！ 全課程包含大量不同的網頁版型範例教學、網頁構圖教學範例...等，上課實際操作每一個範例~現場練習~本課程將從最基本的網頁版型構成觀念，和基本構圖教學講起，課程內將有大量的範例讓您實際操作，因唯有厚實的基本功夫，才能設計出高水準的網頁作品！

Responsive Web Design 手機網頁版型設計班

這個 RWD 教學課程有大量的實作範例讓同學練習，（ RWD 中文翻譯：響應式網頁設計、自適應網頁設計、回應式網頁設計）都是針對 iPhone 和 Android 智慧型手機還有 iPad 和 Android 平板電腦所量身定做的網頁版型設計案例，主要會使用到 HTML5 語法做為網頁的主結構，搭配 CSS3 新語法來排版，對於不同尺寸大小的手機或是平版電腦，設計彈性伸縮的網頁版型，可以讓同一套網頁在電腦上正常瀏覽，遇到平版電腦時會自動轉換成適合平版電腦大小的版型，遇到手機時也會再自動轉換成適合手機大小的版型，做到真正的跨平台網頁設計，最後還會提到 CSS3 新的動態網頁效果...等。
