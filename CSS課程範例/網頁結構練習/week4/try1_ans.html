Dreamweaver & CSS 網頁版型設計班

本課程是最基礎的網頁設計入門教學，使用目前業界網頁設計最專業且最普及的軟體 Dreamweaver，課堂上除了對這個軟體的工具操作方法一一解說之外，更著重於實務應用，規劃以完全〝範例〞導向為主，讓學生在課堂上紮紮實實製作出一系列各式各樣的網頁。此外，課程的教學方向將以業界網頁設計製作流程為主軸，介紹如何規劃網站架構、網站檔案結構規劃和各種網頁編排應用整合…等。

Front-End Engineer 前端工程師養成班

前端工程師是近年來熱門的新興職業，是網頁設計師與程式設計師之間重要的溝通橋樑，更是中大型企業中不可或缺的專業人才。本課程將帶領你進入前端工程師的領域，了解成為前端工程師所具備的條件，讓你從一般的網頁設計師，搖身一變成為搶手的前端工程師。課程中會以目前熱門的前端技術當作練習案例，徹底了解AJAX與JSON整合應用、開發工具除錯、API呼叫與使用、高效能程式碼撰寫、等等專業技巧。
